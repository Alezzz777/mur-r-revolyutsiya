# 🐾 Мур-р-революция

Революция без баррикад — только с мурчанием. Мы не воюем, мы мурчим.

Статичный сайт: котик с [cataas.com](https://cataas.com), манифест и кнопка
«переслать котика». Открыл ссылку → погладил котика → прочёл манифест →
передал дальше. 🐱➡️🐱➡️🐱

## Что внутри

- `index.html` — весь сайт в одном файле (вёрстка, стили, логика).
- `render.yaml` — блупринт для деплоя на [Render](https://render.com) как
  статичного сайта.

## Запустить локально

Просто открой `index.html` в браузере. Для отправки котику нужен интернет
(картинки тянутся с cataas.com), но всё остальное работает офлайн.

## Задеплоить на Render

1. Залить эту папку в Git-репозиторий (GitHub/GitLab).
2. На render.com → **New → Blueprint** → подключить репозиторий.
3. Render прочитает `render.yaml` и поднимет статичный сайт.

## Как это работает

- **Свежий котик** — каждый запрос к cataas.com с уникальным параметром
  отдаёт нового котика.
- **Погладить** — клик по котику запускает «мур-р-р».
- **Переслать** — через `navigator.share` (на телефоне открывается родное
  меню «Поделиться»), на десктопе ссылка копируется в буфер.
- **Красивая карточка при пересылке** — Open Graph теги показывают котика
  и манифест-тизер в превью ссылки.
